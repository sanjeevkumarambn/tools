# Rojgar Sangam Tools

Free online PDF and image tools built for students and government job seekers across India. All tools are browser-based — no file upload to any server, no signup required.

**Live Website:** [tools.rojgarsangam.in](https://tools.rojgarsangam.in)

---

## Available Tools

### PDF Tools
- **Merge PDF** — Combine multiple PDF files into one
- **Split PDF** — Extract pages from a PDF
- **Compress PDF** — Reduce PDF file size
- **Word to PDF** — Convert DOC/DOCX to PDF
- **JPG to PDF** — Convert images to PDF
- **PDF to JPG** — Extract images from PDF

### Photo Tools
- **Image Resizer** — Resize images to exact pixel and KB size
- **Remove Background** — Remove image background automatically
- **Age Calculator** — Calculate exact age from date of birth
- **Name & DOB on Photo** — Add name and date of birth on passport photo
- **Photo Signature Joiner** — Combine passport photo and signature
- **CGPA to Percentage** — Convert CGPA to percentage for job forms

---

## Tech Stack

- HTML, CSS, JavaScript — no frameworks
- Hosted on GitHub Pages + Cloudflare CDN
- PDF processing: PDF-Lib, PDF.js
- Image processing: Canvas API
- No backend, no database, no server

---

## Privacy

All file processing happens entirely inside the user's browser. No file is ever uploaded to any server. No personal data is stored.
